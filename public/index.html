UX Was Here

UX Was Here is a social networking platform that allows users to connect with like minded individuals wanting to share user experience and design ideas. Users can create their own profiles, upload posts with images, connect with others, as well as a messaging system for private conversations. The website was built on React and HTML/CSS/Javascript, and using Firestore Firebase for the back-end database.

I guaranteed open communication with the client and provided guidance to my group of 4 to ensure smooth progress and a successful outcome. I also focused on implementing seamless front-end and back-end communication of the website by utilizing React. This was applied to key pages such as the user profile, messaging, and admin sections to enhance the user experience. Unfortunately the git repository is private for this project.

Pugna Ultima

Pugna Ultima is a 2D multiplayer fighting game created with Unity and Photon Engine's Fusion networking library, with a team of 15 following industry-standard software development practices.

As a co-leader of a sub-group of six, I oversaw the design and development of the core mechanics, including character movement, attacks, and multiplayer integration. I ensured that all of our code met Unity's coding norms by reviewing and integrating contributions from team members and ensuring the seamless integration of scripts and components. By promoting clear communication and adhering to coding standards, I contributed to the successful development of Pugna Ultima. Despite project challenges, we delivered a playable game on time.
