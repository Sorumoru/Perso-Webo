UX Was Here

UX Was Here is a social networking platform that allows users to connect with like minded individuals wanting to share user experience and design ideas. Users can create their own profiles, upload posts with images, connect with others, as well as a messaging system for private conversations. The website was built on React and HTML/CSS, and using Firestore Firebase for the back-end database.

I worked on ensuring a seamless communication with the client as well as guiding my group of 4 during the 3-month long project. I also worked on the user profile page

Unfortunately the repository is private for this project.
